# Task: Student Frontend — Courses

## Description
Build the student-facing course browsing pages: dashboard with progress overview, sections list with progress bars, section detail with topic list, and topic/lesson page with HTML content, YouTube embed, and "mark as complete" functionality. All UI text in Kazakh.

## Acceptance Criteria
- [ ] /student/dashboard — Shows overall progress summary, recent grades, welcome message
- [ ] /student/sections — Displays 5 sections as cards with title, description, icon, and progress bar (% completed topics)
- [ ] /student/sections/:id — Lists topics in section with completion checkmarks, links to topic detail
- [ ] /student/topics/:id — Renders lesson HTML content, YouTube video embed (if video_url exists), lesson images
- [ ] "Тақырыпты аяқтау" button on topic page — marks topic as completed via API, updates progress
- [ ] Progress bars update in real-time after completing topics
- [ ] API integration: api/courses.ts with functions for getSections, getSection, getTopic, completeTopic, getProgress
- [ ] TypeScript interfaces: Section, Topic, Lesson, TopicProgress
- [ ] Responsive design for all pages
- [ ] All text in Kazakh (Бөлімдер, Тақырыптар, Сабақ, Аяқталды, etc.)

## Technical Details
- Pages in pages/student/: Dashboard.tsx, Sections.tsx, SectionDetail.tsx, TopicDetail.tsx
- Reusable components: ProgressBar, SectionCard, TopicList
- API functions in api/courses.ts using the Axios instance from Task 4
- useEffect for data fetching, useState for local state
- YouTube embed: iframe with video_url, 16:9 aspect ratio container
- HTML content rendered with dangerouslySetInnerHTML (content is trusted from teacher/seed)

## Dependencies
- [ ] Task 4 (Auth Frontend) — layout components, auth context, axios instance
- [ ] Task 5 (Course Backend) — API endpoints for courses

## Effort Estimate
- Size: M

## Definition of Done
- [ ] Student can browse all sections and topics
- [ ] Lesson content displays correctly (HTML + video + images)
- [ ] Topic completion works and progress updates
- [ ] Dashboard shows accurate progress summary
- [ ] All UI in Kazakh, responsive layout
