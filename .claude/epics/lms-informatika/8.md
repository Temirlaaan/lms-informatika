# Task: Seed Data & Polish

## Description
Create Django management command `seed_data` that populates the entire course with realistic Kazakh-language content: 5 sections, 15 topics, 15 lessons, 15 quizzes, 75 questions with choices. Create test users. Perform final integration testing of all flows.

## Acceptance Criteria
- [ ] Management command: `python manage.py seed_data` runs successfully
- [ ] Test users created:
  - Superuser: admin / admin123
  - Teacher: teacher / teacher123 (role=teacher)
  - Student: student / student123 (role=student, grade_class="5А")
- [ ] 5 sections created with Kazakh titles matching curriculum:
  1. Ақпарат және ақпаратты ұсыну
  2. Компьютерлік графика
  3. Робототехника негіздері
  4. Роботтың қозғалысы және алгоритмдер
  5. Компьютер және интернет қауіпсіздігі
- [ ] 15 topics (3 per section) with Kazakh titles
- [ ] 15 lessons with realistic HTML content in Kazakh (at least 2-3 paragraphs each)
- [ ] 15 quizzes (one per topic) with title, description, time_limit=15, max_attempts=3
- [ ] 75 questions (5 per quiz) with mixed types: single, multiple, true_false
- [ ] Each question has 4 choices with correct answer(s) marked
- [ ] All content aligned with actual 5th grade Informatics curriculum in Kazakhstan
- [ ] Command is idempotent (can run multiple times without duplicating data)
- [ ] Full student flow tested: register → browse → complete topic → take quiz → view grades
- [ ] Full teacher flow tested: login → view content → view gradebook → view stats

## Technical Details
- Command in courses/management/commands/seed_data.py (or a shared app)
- Use get_or_create for idempotency
- Lesson HTML content: educational text with headers, paragraphs, bold terms, bullet lists
- Question distribution per quiz: ~2 single, ~2 multiple, ~1 true_false
- Content must be factually accurate for 5th grade level
