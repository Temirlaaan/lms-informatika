# Task: Student Frontend — Quiz & Grades

## Description
Build the student quiz-taking experience and grade journal. Quiz page with countdown timer, question navigation, auto-submit on timer expiry, and result page with score breakdown. Grade journal page showing all grades by section.

## Acceptance Criteria
- [ ] /student/quiz/:id — Quiz page: displays questions one at a time or all at once, radio/checkbox for answer selection
- [ ] Countdown timer component: starts from quiz time_limit_minutes, visible countdown, auto-submits when reaching 0
- [ ] Answer selection: radio buttons for 'single' type, checkboxes for 'multiple' type, radio for 'true_false'
- [ ] Submit button: sends all answers to API, navigates to result page
- [ ] Auto-submit: when timer expires, automatically submits current answers
- [ ] /student/quiz/:id/result — Shows score (%), grade (2-5), per-question breakdown (correct/incorrect, correct answer shown)
- [ ] /student/grades — Grade journal: table with sections, quiz scores, grade values, color-coded (5=green, 4=blue, 3=yellow, 2=red)
- [ ] Attempt limit display: show remaining attempts before starting quiz
- [ ] Error handling: "no attempts remaining" message when limit reached
- [ ] API integration: api/quizzes.ts (getQuiz, startQuiz, submitQuiz, getAttempts, getAttemptDetail), api/grades.ts (getMyGrades)
- [ ] All UI text in Kazakh

## Technical Details
- Pages in pages/student/: Quiz.tsx, QuizResult.tsx, Grades.tsx
- Components: Timer (countdown with useEffect + setInterval), QuestionCard, ChoiceSelector, GradeTable
- Timer: useRef for interval ID, cleanup on unmount, auto-submit callback
- Quiz state: track selected answers per question in useState map (questionId → choiceIds[])
- Submit payload: { answers: [{ question_id, selected_choice_ids: [] }] }
- Grade colors via Tailwind: text-green-600, text-blue-600, text-yellow-600, text-red-600

## Dependencies
- [ ] Task 4 (Auth Frontend) — layout, auth context, axios
- [ ] Task 9 (Quiz & Grades Backend) — quiz and grades API endpoints
