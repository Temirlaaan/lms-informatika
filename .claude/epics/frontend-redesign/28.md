# Task: Redesign Student Quiz Pages

## Description
Redesign QuizPage, QuizResultPage, and GradesPage for students using shadcn/ui components. Focus on clear question presentation, timer visibility, and result visualization.

## Acceptance Criteria
- [ ] QuizPage: question Card with number Badge, answer options as selectable Cards/buttons, timer Badge (red when < 60s), Progress bar showing question progress, submit Button with AlertDialog confirmation
- [ ] QuizResultPage: score display with large number and Badge (pass/fail), answer review list with correct/incorrect indicators, retry Button
- [ ] GradesPage: Table component with section names, scores, dates; Badge for grade levels
- [ ] Timer resilience preserved (localStorage-based)
- [ ] All pages work in light and dark themes
- [ ] Mobile-friendly quiz layout (full-width answer options)

## Technical Details
- Quiz answers: use Button variant="outline" with selected state via data attribute or className
- Timer: Badge component with destructive variant when time is low
- Results: consider a simple radial/circular score display with CSS (no chart library needed)
- GradesPage table: use Table, TableHeader, TableRow, etc. from shadcn
- Keep existing quiz logic (start, submit, timer) intact
