# Task: Create Core UI Components

## Description
Create the foundational shadcn/ui components that will be used across all pages: Button, Card, Input, Textarea, Label, Select, Badge, Separator, Skeleton, and Progress. Each component follows shadcn/ui patterns with CVA variants.

## Acceptance Criteria
- [ ] Button component with variants: default, destructive, outline, secondary, ghost, link; sizes: default, sm, lg, icon
- [ ] Card component with CardHeader, CardTitle, CardDescription, CardContent, CardFooter
- [ ] Input component with proper styling and dark mode support
- [ ] Textarea component
- [ ] Label component
- [ ] Select component (using @radix-ui/react-select)
- [ ] Badge component with variants: default, secondary, destructive, outline
- [ ] Separator component
- [ ] Skeleton component for loading states
- [ ] Progress component for progress bars
- [ ] All components use `cn()` from `@/lib/utils`
- [ ] All components support dark mode via CSS variables
- [ ] `npx tsc -b` passes

## Technical Details
- Install @radix-ui/react-select, @radix-ui/react-separator, @radix-ui/react-progress
- Each component exports from its own file in `src/components/ui/`
- Use `React.forwardRef` for all components
- Use CVA for variant management on Button and Badge
- Follow shadcn/ui source code patterns exactly
